# Тестовое задание

## Описание

Решение тестового задания для позиции [Название должности]

## Стек технологий

- Nuxt 3
- TypeScript
- Vue 3
- @nuxt/image

## Установка

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```
